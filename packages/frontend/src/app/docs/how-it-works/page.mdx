# How It Works

Understand how Horizon transforms yield-bearing assets into tradeable tokens.

---

## The Flow

<div className="my-6 rounded-lg border border-border bg-card p-4">
  <Image
    src="/docs/diagrams/protocol-overview.svg"
    alt="Protocol flow: Yield Token → SY → PT + YT"
    width={800}
    height={300}
    className="w-full dark:invert-0"
  />
</div>

1. **Deposit** your yield-bearing token (e.g., nstSTRK)
2. **Wrap** it into Standardized Yield (SY)
3. **Split** SY into Principal Token + Yield Token
4. **Choose:** hold for yield, trade, or provide liquidity

---

## The Core Rule

At any time before maturity:

<Formula display>{"P_{PT} + P_{YT} = 1 \\text{ SY}"}</Formula>

This means you can always:
- **Split:** <Formula>{"1 \\text{ SY} \\to 1 \\text{ PT} + 1 \\text{ YT}"}</Formula>
- **Combine:** <Formula>{"1 \\text{ PT} + 1 \\text{ YT} \\to 1 \\text{ SY}"}</Formula>

---

## At Maturity

<Callout type="warning">
**PT** redeems for full value (1 underlying asset)

**YT** expires worthless — claim any accrued yield before expiry
</Callout>
