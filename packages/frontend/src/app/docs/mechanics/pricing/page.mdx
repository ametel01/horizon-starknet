# Pricing

How PT and YT prices are determined by the AMM.

---

## The Fundamental Equation

At any time before maturity:

<Formula display>{"P_{PT} + P_{YT} = 1 \\text{ SY}"}</Formula>

This relationship is enforced by arbitrage. If prices diverge, traders profit by:
- **Splitting SY** when <Formula>{"P_{PT} + P_{YT} > 1"}</Formula>
- **Combining PT + YT** when <Formula>{"P_{PT} + P_{YT} < 1"}</Formula>

---

## PT Pricing Formula

PT price is derived from the **implied yield** — the annualized return if held to maturity:

<Formula display>{"P_{PT} = \\frac{1}{(1 + r)^{t}}"}</Formula>

Where:
- <Formula>{"r"}</Formula> = implied yield (annualized, e.g., 0.10 for 10%)
- <Formula>{"t"}</Formula> = time to expiry in years (e.g., 0.5 for 6 months)

## YT Pricing (Derived)

Since <Formula>{"P_{PT} + P_{YT} = 1"}</Formula>:

<Formula display>{"P_{YT} = 1 - P_{PT}"}</Formula>

Using the example above:
- <Formula>{"P_{YT} = 1 - 0.9535 = \\textbf{0.0465}"}</Formula>

## AMM Curve: Logit-Based Exchange Rate

The Horizon AMM uses a **logit-based curve** optimized for yield trading:

  <Image

</div>

### The Exchange Rate Formula

<Formula display>{"\\text{exchangeRate} = \\frac{\\text{logit}(p)}{\\text{rateScalar}} + \\text{rateAnchor}"}</Formula>

Where the **logit function** is:

<Formula display>{"\\text{logit}(p) = \\ln\\left(\\frac{p}{1-p}\\right)"}</Formula>

And <Formula>{"p"}</Formula> is the proportion of PT in the pool:

<Formula display>{"p = \\frac{\\text{PT reserve}}{\\text{PT reserve} + \\text{SY reserve}}"}</Formula>

### Key Properties

1. **Rate-based pricing** — Trades move implied yield, not token ratios
2. **Time decay** — Rate scalar increases as maturity approaches, flattening the curve
3. **Dynamic anchor** — Recalculated after each trade to maintain rate continuity
4. **Fee decay** — Swap fees decrease linearly toward expiry

## At Maturity

As expiry approaches:

- <Formula>{"P_{PT} \\to 1.0"}</Formula> (guaranteed redemption value)
- <Formula>{"P_{YT} \\to 0"}</Formula> (no remaining yield rights)
- <Formula>{"r \\to 0"}</Formula> (no time value left)
